list - 목록 만들기

목록의 종류

순서 없는 목록(unordered list) = <ul>
순서 있는 목록(ordered list) = <ol>
정의 목록(description list) = <dl>

1. 순서 없는 목록 = <ul>

  • coffee
  • tea
  • milk
  • web
  • java
  • jsp
  • db

문제. 순서 없는 목록 만들기

  • html
  • css
  • javascript

2. 순서 있는 목록 = <ol>

  1. coffee
  2. tea
  3. milk
  1. web
  2. java
  3. job
  4. jsp
  5. db

문제. 순서 있는 목록 만들기 (빛의 삼원색...)

  1. red = rgb(255,0,0) = #ff0000

  3. blue = rgb(0,0,255) = #0000ff
  4. white = rgb(255,255,255) = #ffffff
  5. black = rgb(0,0,0) = #000000
  6. yellow = rgb(255,255,0) = #ffff00
