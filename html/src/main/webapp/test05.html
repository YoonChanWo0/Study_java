form으로 사용자가 정보전송 가능한 페이지 만들기

form tag의 속성

action="" 사용자가 전송한 정보를 받을 서버파일
method="post/get" 정보를 전송하는 방식

여러 폼요소

   <label for="abc">... </label>     <input type="" id="abc">
   input에 대한 설명문을 label로 표시하며 서로 for의 값과 id값을 일치시킴
  1. <input type="text"> 주관식. 한 줄 글상자

    maxlength="n" 으로 입력값의 최대길이 제한 가능. (size="n"으로 입력상자의 길이 지정 가능.)
  2. <input type="password"> 주관식. 암호 글상자
  3. <input type="checkbox"> 다중 선택형 객관식. 체크박스
    좋아하는 색상은?

    checkbox (checked="checked") 로 초기값으로 설정 가능.
  4. <input type="radio"> 단일 선택형 객관식. 라디오버튼
    (단일 선택이 되려면 같은 질문의 선택지끼리 name의 값을 일치시킴)
    빨강색을 좋아하나요?
    검정색을 좋아하나요?
  5. <input type="hidden"> 사용자에게 안보이는 정보 전송
  6. <input type="file"> 파일 전송용
  7. <textarea> ... </textarea> 주관식. 큰 글상자
    cols="n"와 rows="n" 로 가로 세로 크기 지정 가능
  8. <select> ... </select> 공간 절약형 객관식.
    <option>으로 각 항목들을 묶어줌
    좋아하는 색상은?
    싫어하는 색상은? option에 selected(selected="selected")로 초기값으로 설정 가능.
    가장 자신있는 과목은? select에 size="n" 속성으로 상자형 가능.
    자신있는 과목은? (두개 이상 선택 가능) select에 multiple(multiple="multiple")속성으로 다중 선택형 상자 가능.
    option들을 그룹화하여 묶어 나타내려면
    해당 option들을 optgroup태그로 묶어주고
    label 속성으로 해당 옵션그룹의 제목이나 주제를 나타냄
  9. <input type="submit/reset/button"> 전송/취소/일반 버튼

    value로 버튼의 글자 변경 가능
  10. <input type="image"> 전송용 이미지
    src로 이미지경로를 지정
