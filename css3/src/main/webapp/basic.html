css basic

1. css 적용방식 3가지

  • External CSS - 외부파일방식.
    css소스를 *.css 형식으로 독립시키고 웹문서와 연결해서 적용. (<head>안에 <link href="*.css" rel="stylesheet">)
    여러 웹문서에 통일성 있는 디자인 적용이 가능하며 수정 용이, 유지보수 용이.
  • Internal CSS - 내장방식.
    css소스가 <head>안에 <style>영역에 존재. (<style type="text/css"> ... </style>)
    태그와 분리 되어있으나 웹문서에 종속적이어서 해당 페이지에만 적용됨.
  • Inline CSS - 인라인방식.
    css소스를 해당 태그 바로 뒤에 style="css속성:속성값;"형식으로 적용.
    태그와 붙어있어 소스가 길어지고 수정이 곤란하며 유지보수에 불리하고 통일성 있는 디자인 곤란
    세가지 방식중 적용 CSS우선순위가 가장 높음.

2. css문법

선택자 {속성:속성값;}

3. css 대표 선택자 3가지

  • tag선택자 - 태그 그대로를 선택자로 활용. 강제 적용, 자동 적용.
  • id선택자 - 태그 뒤에 id를 추가해서 id값을 선택자(#id값)로 활용.
    해당 내용이 다른 태그에 적용될 수 없음. 주로 구조적 용도로 이용.
  • class선택자 - 태그 뒤에 class를 추가해서 class값을 선택자(.class값)로 활용.
    class값은 중복 적용이 가능하므로 주로 디자인적 용도로 이용.
    class들을 조합해서 적용하거나 그룹으로 적용 가능.
