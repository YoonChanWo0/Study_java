padding - 요소의 내부여백

1. padding의 방향별 지정속성

  • padding-top: n ; 위에만 n만큼 적용
  • padding-right: n ; 우측에만 n만큼 적용
  • padding-bottom: n ; 아래만 n만큼 적용
  • padding-left: n ; 좌측에만 n만큼 적용

2. padding의 값 갯수별 지정법

  • padding:n ; 양사방 동일하게 적용
  • padding:n n ; 상하, 좌우가 서로 동일하게 적용
  • padding:n n n ; 상, 좌우 동일, 하의 순서
  • padding:n n n n; 상 우 하 좌의 순서
