background 배경

  1. background-color: ; 배경 색상 지정
  2. background-image:url("경로");배경 이미지 지정

background-image의 속성들

  • background-repeat: ; 배경이미지의 반복 지정
    • repeat
    • repeat-x
    • repeat-y
    • no-repeat
  • background-size: ; 배경이미지의 크기 지정
    • cover 이미지 하나로 영역전체 다덮기
      (이미지가 잘려나갈 수도 있음)
    • contain 이미지 하나로 최대한 영역을 다덮기 (이미지가 잘려나가지 않음)
    • Xn Yn 가로세로 크기를 직접 지정
  • background-position: ; 배경이미지의 위치 지정
    • left/center/right top/center/bottom
    • Xn Yn
  • background-attachment: ; 배경이미지의 스크롤여부 지정
    • scroll - 자신영역의 스크롤바에는 고정, 브라우저 스크롤바에는 스크롤.
    • fixed - 자신영역, 브라우저 스크롤바에 모두 고정.
    • local - 자신영역, 브라우저 스크롤바에 모두 스크롤.
